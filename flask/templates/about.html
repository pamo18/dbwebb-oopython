Om denna sida

Denna webbsida är en del av kursen {{ course.get_name() }} hos Blekinge Tekniska Högskola och är uppbyggd med Flask ihop med Jinja2 inom en virtuell miljö, venv, i Python. Jag använder också pakethanteraren PIP för att installera externa python moduler.

Flask är ett microramverk för Python, som gör det möjligt att skapa webbapplikationer. Det är litet men kraftfullt.

Jinja2 är en så kallad “template engine” för Python som renderar template-filer åt oss.

venv är ett sätt att isolera paket från övriga paket installerade på resten av ett system. Detta gör att jag kan installera paket för specifika projekt utan att det krockar med globala installationer eller andra projekts installationer.

PIP är ett pakethanterare för externa python moduler.
